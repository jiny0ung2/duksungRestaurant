# 덕성여대 맛집 소개 웹사이트

이 프로젝트는 덕성여자대학교 주변의 맛집 정보를 소개하는 웹사이트입니다. HTML과 CSS를 사용해 웹페이지를 구성하였으며, 각 맛집에 대한 상세 정보와 맛집 별로 상세 페이지 링크를 제공합니다.

## 개요

- **주제**: 덕성여자대학교 근처의 맛집 소개

## 주요 섹션

- **개요**: 프로젝트 배경 설명
- **맛집 목록**: 다이닝코드 기반 추천 리스트 (210개 중 9개 선별)
- **상세 정보**: 각 음식점의 이미지, 평점, 위치 등을 표 형태로 제공

자세한 정보는 웹페이지 내 테이블에서 확인할 수 있습니다.

## 사용 기술

- HTML5
- CSS3
- 정적 웹페이지 구조

## 페이지 링크

- 미리보기 페이지: `index.html`
- 각 맛집 상세 페이지: `./pages/` 디렉터리에 위치
